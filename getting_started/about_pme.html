About

The Pie Menu Editor (PME) is an add-on for Blender that allows users to create custom menus, dialogs, and various UI elements to streamline their workflow. If you have any questions or need help, please visit the support & community page.

Feature Overview

Pie Menu Editor

Create and customize pie menus with an intuitive graphical interface. Support for multiple layouts, custom icons, and dynamic menus.

Regular Menu Editor
